# Fake News Detection using Machine Learning

## Overview
This project predicts whether a news article is fake or real.

## Tech Stack
- Python
- Pandas
- Scikit-learn
- NLP
- Streamlit

## Workflow
1. Data cleaning
2. Text preprocessing
3. TF-IDF vectorization
4. Logistic Regression training
5. Prediction

## Result
Achieved 96% accuracy.
